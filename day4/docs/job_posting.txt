[주식회사 테크노바 2026년 상반기 신입/경력 채용공고]

■ 회사 소개
테크노바는 2018년 설립된 B2B SaaS 기업으로, 제조업 대상 생산관리 솔루션을 개발합니다.
현재 임직원 180명 규모이며, 본사는 서울시 강남구 테헤란로에 위치합니다.
2025년 시리즈C 투자를 유치했으며, 연매출은 약 420억 원입니다.

■ 모집 부문 1: 백엔드 개발자 (경력 3년 이상)
- 담당 업무: 생산관리 플랫폼 API 설계 및 개발, 대용량 트래픽 처리, 레거시 시스템 마이그레이션
- 필수 자격: Python 또는 Java 기반 서버 개발 경력 3년 이상, RDBMS 설계 및 쿼리 최적화 경험,
  RESTful API 설계 경험, Git 기반 협업 경험
- 우대 사항: Django 또는 Spring Boot 실무 경험, AWS 인프라 운영 경험, Docker/Kubernetes 사용 경험,
  MSA 전환 프로젝트 참여 경험
- 채용 인원: 3명

■ 모집 부문 2: AI 엔지니어 (신입 및 경력 무관)
- 담당 업무: LLM 기반 사내 문서 검색 시스템 구축, RAG 파이프라인 설계, 프롬프트 최적화
- 필수 자격: Python 능숙, 머신러닝 기초 지식 보유, 학사 이상 학위
- 우대 사항: LangChain/LangGraph 사용 경험, 벡터 데이터베이스(FAISS, Pinecone 등) 활용 경험,
  논문 게재 또는 오픈소스 기여 이력, 영어 논문 독해 가능
- 채용 인원: 2명

■ 모집 부문 3: 프로덕트 디자이너 (경력 2년 이상)
- 담당 업무: B2B 대시보드 UI/UX 설계, 디자인 시스템 구축 및 운영
- 필수 자격: Figma 능숙, B2B 또는 데이터 시각화 프로덕트 디자인 경험 2년 이상
- 우대 사항: 프론트엔드 기초 지식(HTML/CSS), 사용자 리서치 수행 경험
- 채용 인원: 1명
